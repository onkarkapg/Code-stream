# 🚀 CodeStream - Modern Coding Platform UI

## 📌 Overview
**CodeStream** is a modern, responsive landing page for an online coding education platform. It showcases courses, bootcamps, and developer resources with an attractive UI and clean design.

This project is built using **HTML and CSS only**, making it lightweight and beginner-friendly.

---

## ✨ Features

- 🎯 Clean and modern UI design  
- 📱 Fully responsive layout  
- 🚀 Hero section with call-to-action  
- 📊 Statistics section (Developers, Courses, etc.)  
- 📚 Bootcamp cards with hover effects  
- 🔥 Styled navigation bar  
- 🌙 Elegant footer with contact info  
- 💡 Smooth hover animations and gradients  

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- Flexbox & Grid Layout
- Responsive Design Principles
